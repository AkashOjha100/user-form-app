# User Form App

A simple React application for user forms.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Docker

To run with Docker:

1. Build the image: `docker build -t user-form-app .`
2. Run the container: `docker run -p 3000:3000 user-form-app`

## CI/CD

This project uses GitHub Actions for CI/CD with Docker.
